Informações do Webservice

Base URL

http://localhost:8080/api

1. Editar Pessoa

Rota: /pessoas/{id}

Método: PUT

Exemplo de requisição:

{
    "nome": "Maria Silva",
    "dataNascimento": "1985-04-15",
    "cpf": "987.654.321-00",
    "funcionario": true,
    "gerente": false
}

Descrição: Atualiza as informações de uma pessoa com o ID especificado.

2. Listar Pessoas

Rota: /pessoas

Método: GET

Descrição: Retorna a lista de todas as pessoas cadastradas.

3. Obter Pessoa

Rota: /pessoas/{id}

Método: GET

Descrição: Retorna as informações de uma pessoa com o ID especificado.

4. Cadastrar Pessoa

Rota: /pessoas

Método: POST

Exemplo de requisição:

{
    "nome": "Maria Souza",
    "dataNascimento": "1985-04-15",
    "cpf": "987.654.321-00",
    "funcionario": false,
    "gerente": true
}

Descrição: Cadastra uma nova pessoa.

1. Adicionar Membro ao Projeto

Rota: /projetos/{id}/members

Método: POST

Exemplo de requisição:

{
    "id": 1
}

Descrição: Adiciona um membro ao projeto com o ID especificado.

2. Remover Membro do Projeto

Rota: /projetos/{id}/members

Método: DELETE

Exemplo de requisição:

{
    "id": 1
}

Descrição: Remove um membro do projeto com o ID especificado.

3. Listar Projetos

Rota: /projetos

Método: GET

Descrição: Retorna a lista de todos os projetos cadastrados.

4. Obter Projeto

Rota: /projetos/{id}

Método: GET

Descrição: Retorna as informações de um projeto com o ID especificado.

5. Cadastrar Projeto

Rota: /projetos

Método: POST

Exemplo de requisição:

{
"nome": "Projeto Alpha",
"dataInicio": "2024-01-01",
"dataPrevisaoFim": "2024-12-31",
"descricao": "Descrição do Projeto Alpha.",
"status": "PLANEJADO",
"orcamento": 150000.00
}

Descrição: Cadastra um novo projeto.

Valores possiveis para status: EM_ANALISE, ANALISE_REALIZADA, ANALISE_APROVADA, INICIADO, PLANEJADO, EM_ANDAMENTO, ENCERRADO, CANCELADO

6. Editar Projeto

Rota: /projetos/{id}

Método: PUT

Exemplo de requisição:

{
    "nome": "Projeto Alpha",
    "dataInicio": "2024-01-01T00:00:00.000+00:00",
    "dataPrevisaoFim": "2024-12-31T00:00:00.000+00:00",
    "dataFim": "2024-12-31T00:00:00.000+00:00",
    "descricao": "Descrição do Projeto Alpha.",
    "status": "PLANEJADO",
    "orcamento": 150000.0
}

Descrição: Atualiza as informações de um projeto com o ID especificado.

7. Deletar Projeto

Rota: /projetos/{id}

Método: DELETE

Descrição: Deleta um projeto com o ID especificado.
